1. Сначала запустить VS Studio
2. Выбрать открыть проект или решение
3. Зайти по путь Реверсивная диаграмма Ганта/Backend
4. Выбрать файл Backend.sln
5. Нажать открыть
6. Чут чут подождать
7. Вверху нажать запустить (▶ http)
8. Если вылезет соглашение, то нажать Ок
9. Подождать, пока откроется браузер
10. Затем открыть файл во Frontend index.html
